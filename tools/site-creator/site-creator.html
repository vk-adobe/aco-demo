Create your Commerce Storefront

This tool provisions your own version of the aemshop.net storefront. It will clone the code and content behind aemshop.net to your own code and content repositories and publish them on your behalf.

Note: This tool can only create one fork per template repository per GitHub user account. If you already have a fork of the Boilerplate, you will need delete it to use this tool. For more granular control, you can use the Commerce Scaffolding CLI or go through the onboarding tutorial here.
